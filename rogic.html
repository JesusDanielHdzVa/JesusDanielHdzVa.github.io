¿Rogic?

¿Que es?

stroRogic (Rule-based Open Graph Interactive Computing) es un lenguaje de programación y un sistema basado en reglas para el procesamiento de datos y la inteligencia artificial. Se enfoca en la representación del conocimiento y la inferencia lógica a través de grafos abiertos, lo que lo hace útil en áreas como la IA simbólica, la toma de decisiones y la automatización de procesos.

¿Porque Aprenderlo?

Ideal para IA simbólica: Facilita la representación y manipulación del conocimiento en aplicaciones de inteligencia artificial.

Automatización de reglas: Permite definir reglas lógicas para automatizar procesos y tomar decisiones basadas en datos estructurados.

Interoperabilidad con grafos: Se integra bien con sistemas que usan bases de datos gráficas como Neo4j.

Útil en razonamiento automático: Aplicable en sistemas expertos, análisis de datos y generación de inferencias.

Print

Para imprimir un mensaje en la consola, se usa la función print():

print("¡Hola, mundo!")

El comando print nos servira para muchas cosas ya que junto a este se nos permitira imprimir cualquier tipo de cosa mediante la terminal que nos da visual estudio code


Mas ejemplos de este tipo

Aqui se imprimira el nombre "Jesús"

print("Hola me llamo Jesús")

Aqui se respondera a una suma

print("El resultado de tu suma es >= 10")

Aqui preguntaremos al usuario si es que necesita algo mas

print("Necesitas algo mas?")

Variables en Python

Python permite definir variables sin especificar su tipo:

nombre = "Jesús" edad = 20 precio = 19.99

Estructuras de control

Condicionales

Ejemplo de estructura if:

edad = 18 if edad >= 18: print("Eres mayor de edad") else: print("Eres menor de edad")

Bucles

Ejemplo de un bucle for:

for i in range(5): print("Iteración", i)

Conclusión

Python es un lenguaje poderoso y fácil de aprender. Es ideal tanto para principiantes como para desarrolladores avanzados.
