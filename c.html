¿C++?

¿Que es?

C++ es un lenguaje de programación de propósito general que se originó como una extensión de C, agregando características como la programación orientada a objetos. Es ampliamente utilizado en el desarrollo de software de alto rendimiento, sistemas operativos, motores de videojuegos, aplicaciones financieras y software embebido.

¿Porque Aprenderlo?

Alto rendimiento: Es un lenguaje eficiente y rápido, ideal para aplicaciones que requieren un uso intensivo de recursos.

Versatilidad: Se usa en diversas áreas como desarrollo de videojuegos, inteligencia artificial, sistemas operativos y más.

Control sobre hardware: Permite manipular memoria y recursos del sistema de manera más precisa que otros lenguajes de alto nivel.

Base para otros lenguajes: Aprender C++ facilita la comprensión de otros lenguajes como C, C#, Java y Rust.

Variables en C++

En C++, las variables son contenedores que almacenan valores. Cada ejemplo muestra la declaración de una variable y su uso básico.

// Ejemplo 1: Variable entera // Se declara una variable entera 'edad' y se muestra su valor en consola. int edad = 30; std::cout << "Edad: " << edad << std::endl;

Explicación: Se crea una variable de tipo int para almacenar la edad y se imprime su contenido.

// Ejemplo 2: Variable flotante // Se declara una variable de tipo float para almacenar la estatura. float estatura = 1.75; std::cout << "Estatura: " << estatura << std::endl;

Explicación: Se define un valor decimal para representar la estatura y se imprime.

// Ejemplo 3: Variable de tipo carácter // Se declara una variable de tipo char para almacenar una letra. char inicial = 'A'; std::cout << "Inicial: " << inicial << std::endl;

Explicación: Se utiliza char para guardar un solo carácter y se muestra por consola.

// Ejemplo 4: Variable de tipo cadena // Se declara una variable de tipo std::string para almacenar una cadena de texto. std::string nombre = "Carlos"; std::cout << "Nombre: " << nombre << std::endl;

Explicación: Se crea una variable de texto para guardar el nombre y se imprime.

// Ejemplo 5: Variable booleana // Se declara una variable booleana para determinar si alguien es estudiante. bool esEstudiante = true; std::cout << "Es estudiante: " << (esEstudiante ? "Sí" : "No") << std::endl;

Explicación: Se utiliza un valor booleano y el operador ternario para mostrar "Sí" o "No" según corresponda.

Entrada y salida de datos en C++

En C++ se usan std::cin para la entrada y std::cout para la salida de datos. Cada ejemplo muestra cómo interactuar con el usuario.

// Ejemplo 1: Leer un entero // Se solicita al usuario que ingrese un número entero y luego se muestra. #include int main() { int numero; std::cout << "Introduce un número entero: "; std::cin >> numero; std::cout << "Número ingresado: " << numero << std::endl; return 0; }

Explicación: Se lee un número entero desde la consola y se imprime en pantalla.

// Ejemplo 2: Leer un número flotante // Se solicita un valor decimal al usuario y se muestra en consola. #include int main() { float valor; std::cout << "Introduce un número con decimales: "; std::cin >> valor; std::cout << "Valor ingresado: " << valor << std::endl; return 0; }

Explicación: Se utiliza float para leer y mostrar un número con punto decimal.

// Ejemplo 3: Leer una cadena sin espacios // Se solicita al usuario ingresar una palabra y se muestra. #include #include int main() { std::string palabra; std::cout << "Introduce una palabra: "; std::cin >> palabra; std::cout << "Palabra ingresada: " << palabra << std::endl; return 0; }

Explicación: Se lee una cadena sin espacios utilizando std::cin y se imprime.

// Ejemplo 4: Salida de múltiples datos // Se definen dos variables y se muestran ambas en una sola línea. #include int main() { int x = 10; float y = 3.14; std::cout << "Valor de x: " << x << ", valor de y: " << y << std::endl; return 0; }

Explicación: Se muestran múltiples variables en una salida formateada.

// Ejemplo 5: Leer una línea completa de texto // Se utiliza std::getline para leer una línea completa que puede contener espacios. #include #include int main() { std::string frase; std::cout << "Introduce una frase: "; std::cin.ignore(); // Limpia el buffer std::getline(std::cin, frase); std::cout << "Frase ingresada: " << frase << std::endl; return 0; }

Explicación: Permite leer una línea completa de texto, incluyendo espacios, usando std::getline.

Condicionales en C++

Las condicionales permiten ejecutar bloques de código según se cumplan ciertas condiciones. Cada ejemplo ilustra un tipo distinto de estructura condicional.

// Ejemplo 1: Uso básico de if // Se verifica si la variable 'a' es mayor que 0 y, de ser cierto, se imprime un mensaje. #include int main() { int a = 5; if(a > 0) { std::cout << "a es positivo." << std::endl; } return 0; }

Explicación: Se usa la sentencia if para evaluar una condición simple.

// Ejemplo 2: if-else // Se evalúa si 'b' es no negativo; de lo contrario, se indica que es negativo. #include int main() { int b = -3; if(b >= 0) { std::cout << "b es no negativo." << std::endl; } else { std::cout << "b es negativo." << std::endl; } return 0; }

Explicación: Se utiliza la estructura if-else para tomar una de dos decisiones según la condición.

// Ejemplo 3: if-else if-else // Se evalúa el valor de 'c' para determinar si es positivo, negativo o cero. #include int main() { int c = 0; if(c > 0) { std::cout << "c es positivo." << std::endl; } else if(c < 0) { std::cout << "c es negativo." << std::endl; } else { std::cout << "c es cero." << std::endl; } return 0; }

Explicación: Se utiliza if con else if y else para cubrir todas las posibilidades.

// Ejemplo 4: Condicional anidado // Se verifica si 'd' es positivo y, dentro de esa condición, se comprueba si es par o impar. #include int main() { int d = 8; if(d > 0) { if(d % 2 == 0) { std::cout << "d es positivo y par." << std::endl; } else { std::cout << "d es positivo pero impar." << std::endl; } } return 0; }

Explicación: Se usan condicionales anidados para evaluar más de una condición.

// Ejemplo 5: Operador ternario // Se utiliza el operador ternario para imprimir si 'e' es par o impar en una sola línea. #include int main() { int e = 15; std::cout << (e % 2 == 0 ? "e es par." : "e es impar.") << std::endl; return 0; }

Explicación: Se simplifica la evaluación de una condición mediante el operador ternario.

Bucles en C++

Los bucles permiten repetir la ejecución de un bloque de código. Cada ejemplo muestra un tipo distinto de bucle.

// Ejemplo 1: Bucle for simple // Se utiliza un bucle for para iterar 5 veces e imprimir el número de iteración. #include int main() { for (int i = 0; i < 5; i++) { std::cout << "Iteración " << i << std::endl; } return 0; }

Explicación: El bucle for recorre un rango de valores y ejecuta el bloque interno en cada iteración.

// Ejemplo 2: Bucle while // Se usa un bucle while para contar hasta 5, incrementando la variable 'contador'. #include int main() { int contador = 0; while(contador < 5) { std::cout << "Contador: " << contador << std::endl; contador++; } return 0; }

Explicación: El bucle while evalúa la condición antes de cada iteración y continúa mientras sea verdadera.

// Ejemplo 3: Bucle do-while // Se utiliza do-while para ejecutar el bloque al menos una vez y luego verificar la condición. #include int main() { int i = 0; do { std::cout << "Do-while iteración " << i << std::endl; i++; } while(i < 5); return 0; }

Explicación: El bucle do-while garantiza que el bloque se ejecute por lo menos una vez, verificando la condición al final.

// Ejemplo 4: Bucle for con break // Se recorre un bucle for e interrumpe la iteración cuando i alcanza 5. #include int main() { for (int i = 0; i < 10; i++) { if(i == 5) { break; // Sale del bucle } std::cout << "Valor: " << i << std::endl; } return 0; }

Explicación: Se demuestra el uso de break para terminar un bucle de forma anticipada.

// Ejemplo 5: Bucle for-each con std::vector // Se utiliza un bucle for-each para recorrer y mostrar cada elemento de un vector. #include #include int main() { std::vector numeros = {1, 2, 3, 4, 5}; for(auto num : numeros) { std::cout << "Número: " << num << std::endl; } return 0; }

Explicación: El bucle for-each simplifica la iteración a través de todos los elementos de un contenedor como std::vector.

Funciones en C++

Las funciones permiten encapsular bloques de código que realizan tareas específicas. Cada ejemplo muestra distintos aspectos de las funciones.

// Ejemplo 1: Función sin parámetros // Se define una función que imprime un saludo y se llama desde el main. #include void saludar() { std::cout << "¡Hola, mundo!" << std::endl; } int main() { saludar(); return 0; }

Explicación: Se crea una función sencilla que no recibe parámetros y se invoca para mostrar un mensaje.

// Ejemplo 2: Función con parámetros que retorna un entero // La función 'sumar' recibe dos números, los suma y retorna el resultado. #include int sumar(int a, int b) { return a + b; } int main() { std::cout << "Suma: " << sumar(4, 6) << std::endl; return 0; }

Explicación: Se demuestra cómo pasar argumentos a una función y devolver un valor.

// Ejemplo 3: Función que imprime un mensaje // Se define una función que no retorna valor y simplemente imprime un mensaje. #include void mensaje() { std::cout << "Este es un mensaje de la función." << std::endl; } int main() { mensaje(); return 0; }

Explicación: La función se usa para encapsular un mensaje que se muestra al ejecutarse.

// Ejemplo 4: Función recursiva para calcular el factorial // La función 'factorial' se llama a sí misma para calcular el factorial de un número. #include int factorial(int n) { if(n <= 1) return 1; else return n * factorial(n - 1); } int main() { std::cout << "Factorial de 5: " << factorial(5) << std::endl; return 0; }

Explicación: Se utiliza la recursión para calcular el factorial de un número.

// Ejemplo 5: Función con parámetros por defecto // La función 'imprimirMensaje' tiene un parámetro con un valor por defecto, // permitiendo llamarla con o sin un argumento personalizado. #include void imprimirMensaje(std::string msg = "Mensaje por defecto") { std::cout << msg << std::endl; } int main() { imprimirMensaje("Mensaje personalizado"); imprimirMensaje(); return 0; }

Explicación: Se muestra cómo asignar valores por defecto a los parámetros de una función.

Arreglos en C++

Los arreglos permiten almacenar múltiples elementos del mismo tipo. Cada ejemplo ilustra una forma de declarar y utilizar arreglos.

// Ejemplo 1: Arreglo de enteros // Se declara un arreglo de 5 enteros y se recorren para imprimir cada elemento. #include int main() { int numeros[5] = {10, 20, 30, 40, 50}; for (int i = 0; i < 5; i++) { std::cout << "Elemento " << i << ": " << numeros[i] << std::endl; } return 0; }

Explicación: Se crea un arreglo fijo y se utiliza un bucle for para recorrer e imprimir sus elementos.

// Ejemplo 2: Arreglo de flotantes // Se declara un arreglo de números decimales y se imprime cada valor. #include int main() { float temperaturas[4] = {36.6, 37.0, 36.8, 37.2}; for (int i = 0; i < 4; i++) { std::cout << "Temperatura " << i << ": " << temperaturas[i] << std::endl; } return 0; }

Explicación: Se muestra el uso de un arreglo para almacenar valores flotantes y recorrerlo con un bucle.

// Ejemplo 3: Arreglo de cadenas // Se declara un arreglo de strings para almacenar nombres de frutas y se recorre para imprimirlos. #include #include int main() { std::string frutas[3] = {"Manzana", "Banana", "Naranja"}; for (int i = 0; i < 3; i++) { std::cout << "Fruta " << i << ": " << frutas[i] << std::endl; } return 0; }

Explicación: Se utiliza un arreglo de tipo std::string para almacenar y mostrar nombres de frutas.

// Ejemplo 4: Arreglo multidimensional (matriz 2x3) // Se declara una matriz de 2 filas y 3 columnas y se recorren ambos índices para imprimir cada elemento. #include int main() { int matriz[2][3] = { {1, 2, 3}, {4, 5, 6} }; for (int i = 0; i < 2; i++) { for (int j = 0; j < 3; j++) { std::cout << "Elemento [" << i << "][" << j << "]: " << matriz[i][j] << std::endl; } } return 0; }

Explicación: Se declara una matriz bidimensional y se utiliza un bucle anidado para recorrer cada posición.

// Ejemplo 5: Uso de std::vector (arreglo dinámico) // Se declara un vector de enteros y se usa un bucle for-each para imprimir sus elementos. #include #include int main() { std::vector numeros = {5, 10, 15, 20}; for(auto n : numeros) { std::cout << "Número: " << n << std::endl; } return 0; }

Explicación: Se utiliza std::vector para crear un arreglo dinámico y recorrerlo con un bucle de rango.

Ejercicios por Tema

Variables

1. Crea un programa que guarde tu nombre, edad y altura, y luego imprima esos valores.

2. Declara tres variables numéricas (int) y realiza operaciones básicas con ellas (suma, resta, multiplicación, división).

3. Intercambia el valor de dos variables usando una variable auxiliar.

Entrada y salida de datos

1. Pide al usuario su nombre y edad, y muestra un mensaje con esos datos.

2. Solicita dos números al usuario y muestra su suma, resta, multiplicación y división.

3. Pide al usuario su año de nacimiento y muestra su edad actual.

Condicionales

1. Pide al usuario un número y muestra si es positivo, negativo o cero.

2. Solicita la calificación de un estudiante y muestra si aprueba (mayor o igual a 60) o no.

3. Crea un programa que compare dos números e indique cuál es mayor, o si son iguales.

Bucles

1. Muestra los números del 1 al 10 usando un bucle for.

2. Pide un número al usuario y muestra su tabla de multiplicar del 1 al 10.

3. Suma los números del 1 al 100 usando un bucle while.

Funciones

1. Crea una función que reciba dos números y devuelva su suma.

2. Haz una función que determine si un número es par o impar.

3. Crea una función que imprima un mensaje de saludo personalizado.

Arreglos

1. Declara un arreglo de 5 enteros, asigna valores y muestra cada uno con un bucle.

2. Solicita 5 números al usuario, guárdalos en un arreglo e imprime el promedio.

3. Crea un programa que busque el número mayor dentro de un arreglo de 10 elementos.
