¿RoboPro?

¿Que es?

Robo Pro Coding es un entorno de programación utilizado para programar robots y sistemas automatizados, especialmente en el ámbito educativo y de formación técnica. Es desarrollado por fischertechnik y permite programar robots mediante bloques gráficos o código, facilitando el aprendizaje de lógica de programación y control de hardware.

¿Porque Aprenderlo?

Fomenta el pensamiento lógico : Ayuda a desarrollar habilidades de resolución de problemas y razonamiento estructurado.

Introducción a la robótica y automatización: Ideal para aprender sobre sensores, motores y controladores.

Facilidad de uso: Su interfaz visual es intuitiva, permitiendo a principiantes comprender la programación sin necesidad de conocimientos avanzados.

Preparación para el futuro : La automatización y la robótica son tendencias clave en la industria y aprender a programarlas es una ventaja.

Creatividad y experimentación : Permite diseñar y programar robots personalizados, fomentando la innovación.

Print

Para imprimir un mensaje en la consola, se usa la función print():

print("¡Hola, mundo!")

El comando print nos servira para muchas cosas ya que junto a este se nos permitira imprimir cualquier tipo de cosa mediante la terminal que nos da visual estudio code


Mas ejemplos de este tipo

Aqui se imprimira el nombre "Jesús"

print("Hola me llamo Jesús")

Aqui se respondera a una suma

print("El resultado de tu suma es >= 10")

Aqui preguntaremos al usuario si es que necesita algo mas

print("Necesitas algo mas?")
